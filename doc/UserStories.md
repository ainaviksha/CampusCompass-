# User Stories: Master Engineering Application

## Epic: Application Form (Page 1)
1.  **US-01**: As a **Student**, I want to enter my personal and contact details so that colleges can reach me.
2.  **US-02**: As a **Student**, I want to verify my phone number via OTP so that my application is authenticated.
3.  **US-03**: As a **Student**, I want to input my JEE Mains Percentile and other exam ranks/percentiles so that my eligibility and college recommendations are accurate.
4.  **US-04**: As a **Student**, I want to optionally provide my Board and Olympiad details to strengthen my profile.
5.  **US-05**: As a **Student**, I want real-time validation on my inputs so that I don't submit incorrect data.

## Epic: College Discovery (Page 2)
6.  **US-06**: As a **Student**, I want to see colleges categorized (e.g., "Recommended", "Elite") so that I can easily find what fits my profile.
7.  **US-07**: As a **Student**, I want to view colleges in a horizontal scrolling list ("Netflix-style") to quickly browse options without navigating away.
8.  **US-08**: As a **Student**, I want to tap a college card to select it for application so that I can build my list easily.
9.  **US-09**: As a **Student**, I want to expand a category row to see all colleges in a grid view if I want to explore deeper.

## Epic: Decision Making (AI Summary)
10. **US-10**: As a **Student/Parent**, I want to click "Learn More" on my selected colleges to get a detailed summary of their ROI, Placements, and Fees.
11. **US-11**: As a **Student**, I want to see the "Institute Age" and "Batch Size" to understand the maturity of the program.
12. **US-12**: As a **Student**, I want to see noteworthy alumni and key achievements to gauge the prestige of the institute.

## Epic: Checkout & Application
13. **US-13**: As a **Student**, I want to see a floating action bar showing the number of selected colleges so that I know my cart size.
14. **US-14**: As a **Student**, I want to click "Apply Now" to see a breakdown of total fees for all selected colleges.
15. **US-15**: As a **Student**, I want to pay for all applications in a single transaction to save time and effort.
